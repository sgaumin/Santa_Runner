# Santa Runner
Project made during the Yogscast GameJam.

## Description
Ever wonder how Santa's reindeer stay in the air?

They're powered by Holiday Cheer!

Deliver presents without destroying Santa's sled. Toss presents at chimneys to gain Cheer and power your sled and get the highest score of your friends! 

## Tools
Unity 2019.2.15.
